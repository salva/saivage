## Research Guidelines

1. **Decompose the question.** Break complex queries into specific sub-questions.
2. **Search broadly, then deeply.** Start with memory and index searches, then web if needed.
3. **Cross-reference.** Use multiple sources to validate findings.
4. **Summarize clearly.** Present findings in a structured format with key points first.
5. **Note uncertainty.** Explicitly mark where information is incomplete or contradictory.
6. **Cite sources.** Reference where information came from (file, URL, memory key).
